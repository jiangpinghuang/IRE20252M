# knowledge

本目录存放 ChatREQ、基准任务和论文研究可复用的知识资产。

## 内容范围

- `standards/`：需求工程标准、规范、指南和约束说明。
- `srs/`：软件需求规格说明书案例和相关结构化材料。
- `papers/`：与知识库建设相关的论文资料。
- `books/`：课程和研究使用的书籍资料索引。
- `extraction/`：知识抽取、清洗、转换和索引流程。
- `misc/`：领域知识、检索、嵌入、知识图谱和本体等补充资产。

## 维护规则

- 知识材料应记录来源、版本、适用范围和使用限制。
- 可用于检索或实验的材料应保留结构化元数据。
- 不提交敏感、私有或授权不清的数据。
