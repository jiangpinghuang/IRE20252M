# IRE Papers

This directory stores paper assets for Intelligent Requirements Engineering.

## ACM Template Policy

All ACM-style papers must use the provided ACM template in `docs/acmart-primary/`. Do not create or import another ACM template unless the project explicitly upgrades the primary template.

The reusable skeleton in `group-papers/template-paper/` references:

- `../../../docs/acmart-primary/acmart.cls`
- `../../../docs/acmart-primary/ACM-Reference-Format.bst`

## Directory Overview

| Path | Purpose |
| --- | --- |
| `templates/` | Shared writing templates and checklists. |
| `master-paper/` | Integrated ChatREQ TOSEM/TSE outline and future master paper assets. |
| `group-papers/` | Group-level ACM SIGCONF paper skeletons and drafts. |
| `related-work/` | Literature notes and comparison matrices. |
| `figures/` | Shared paper figures. |
| `tables/` | Shared paper tables. |
| `algorithms/` | Algorithm pseudocode and reusable method descriptions. |
| `refs/` | Shared bibliography assets. |
| `writing-guidelines/` | Writing conventions for ICSE/ASE/FSE/TOSEM/TSE-style papers. |
| `paper-assets/` | Paper-ready artifacts such as appendices, checklists, and reproducibility packages. |

## Paper Quality Expectations

Papers should include explicit research questions, baselines, ablation studies, threats to validity, reproducibility notes, and benchmark-oriented evaluation.
