# Roadmap

This roadmap organizes the long-term development of Intelligent Requirements Engineering (IRE), ChatREQ, IRE-Bench, IRE-Datasets, IRE-Knowledge, and IRE-Evaluation.

## 2026

- Build ChatREQ v1 as the initial knowledge-driven multi-agent requirements engineering system.
- Organize the graduate course around eight research groups and reusable project assets.
- Complete eight group modules covering elicitation, environment awareness, requirement graphs, conflict checking, human-in-the-loop adaptation, feasibility validation, architecture alignment, and executable requirements.
- Construct IRE-Bench v1 with shared benchmark tasks, baselines, metrics, and reproducibility protocols.
- Prepare ICSE/ASE-style papers from group modules and the first integrated ChatREQ paper.

## 2027

- Improve knowledge-driven collaboration among specialized agents.
- Build reusable datasets for clarification dialogues, SRS documents, requirement graphs, conflict cases, architecture mappings, and executable tasks.
- Improve the evaluation framework with stronger baselines, ablation studies, human effort metrics, and cross-domain experiments.
- Prepare TOSEM/TSE submissions by extending conference papers with broader theory, more data, deeper evaluation, and stronger validity analysis.

## 2028

- Open-source a stable ChatREQ platform with documented installation, reproducible experiments, and teaching-ready modules.
- Release a public benchmark with task definitions, data splits, baselines, evaluation scripts, and paper-ready reporting templates.
- Establish long-term IRE infrastructure that supports continuing student cohorts and multi-year research.
- Build a cross-domain intelligent RE ecosystem covering business systems, cyber-physical systems, AI-enabled systems, and safety-critical requirements.
