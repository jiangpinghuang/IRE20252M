# docs

本目录存放课程与研究项目的通用文档，包括课程说明、课件、进度安排、阅读材料、协作规范和模板。

## 内容范围

- `syllabus/`：课程大纲、学习目标和课程规则。
- `slides/`：授课课件、讨论材料和演示文档。
- `schedules/`：周次安排、里程碑和关键日期。
- `reading-list/`：论文、书籍和标准的阅读清单。
- `templates/`：课程报告、项目文档和协作模板。

## 维护规则

- 文档应短小清晰，避免重复说明。
- 与小组研究直接相关的材料应放入 `groups/`。
- 与论文投稿直接相关的模板应放入 `papers/templates/`。
