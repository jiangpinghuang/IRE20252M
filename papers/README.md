# papers

本目录存放课程与研究项目的共享论文资产，包括模板、参考文献和主论文材料。

## 内容范围

- `templates/`：ACM 模板、小组论文骨架、课程论文模板和写作规范。
- `references/`：共享 BibTeX、相关工作笔记和文献整理材料。
- `master-paper/`：ChatREQ/IRE 集成论文的大纲、图表、算法和附录资产。

## 维护规则

- 小组论文草稿优先放在对应 `groups/G*/paper/`。
- 共享模板和共用文献放在本目录，避免在各组重复拷贝。
- 论文材料不得包含未经验证的实验结果或虚构结论。
- ACM 论文优先使用 `papers/templates/acmart-primary/` 和 `papers/templates/group-paper/`。
