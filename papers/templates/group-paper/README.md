# group-paper

本目录提供小组论文可复用的 ACM SIGCONF 风格 LaTeX 骨架。

## 内容范围

- `main.tex`：论文入口文件，包含元数据、摘要、关键词、章节和参考文献。
- `sections/`：按软件工程论文结构拆分的章节文件。
- `figures/`：论文图。
- `tables/`：论文表。
- `algorithms/`：算法或方法片段。
- `refs/references.bib`：论文参考文献。

## 模板规则

- 文档类使用 `../acmart-primary/acmart`。
- 参考文献样式使用 `../acmart-primary/ACM-Reference-Format`。
- 小组正式草稿应复制到对应 `groups/G*/paper/` 后维护。
