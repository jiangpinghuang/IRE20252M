# IRE20252M 智能需求工程

本仓库用于组织 IRE20252M 研究生课程的课程资料、科研协作、实验基准、知识资料和各小组工作产物。

## 内容组织

- `docs/`：课程说明、课件、日程和阅读材料。
- `papers/`：论文模板、参考资料和课程公共论文材料。
- `benchmark/`：任务、指标、基线和结果记录。
- `knowledge/`：标准、SRS、论文、书籍和抽取资料。
- `system/`：课程相关系统与工具材料。
- `groups/`：8 个研究小组的论文、报告、实验、数据和代码。
- `ai-resources/`：AI 使用相关提示词、流程、规范和教程。

## 基本规则

按目录职责存放文件；不要在无关目录混放材料。论文正文和实验数据由各组在对应目录内维护，公共资料放入一级公共目录。提交前请确认没有加入临时文件、缓存文件和未授权的大型数据。
