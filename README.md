# IRE20252M

本仓库用于 **Intelligent Requirements Engineering** 硕士课程与长期研究资产管理，面向智能需求工程、知识驱动多智能体系统、AI 原生软件工程、基准构建、数据集积累与论文写作。

仓库不是一次性作业集合，而是多届学生和研究小组共同演化的研究型工作区。

## 目录概览

| 目录 | 用途 |
| --- | --- |
| `docs/` | 课程文档、教学资料、进度安排、阅读材料和通用模板。 |
| `papers/` | 论文模板、参考文献、主论文材料和共享写作资产。 |
| `benchmark/` | 基准任务、指标、基线、实验配置和结果记录。 |
| `knowledge/` | 标准、SRS、论文、书籍、抽取流程和其他知识资产。 |
| `system/` | 系统实现与工程资产，目前核心系统为 `ChatREQ`。 |
| `groups/` | 八个研究小组的论文、报告、实验、数据、代码和笔记。 |
| `ai-resources/` | AI 辅助研究与工程的提示词、流程、指南和参考资料。 |

## 组织原则

- README 优先：重要目录先说明用途、内容和维护规则。
- 真实积累：不写虚假实验、虚假数据或无来源结论。
- 保守归档：不确定归属的材料先保留，并放入最接近的研究目录。
- 组内闭环：小组相关论文、实验、数据、图表、代码和笔记放在对应 `groups/G*/` 下。
- 可复现优先：基准、实验和结果必须记录数据版本、配置、指标和运行方式。

## 长期目标

本仓库服务于可持续的智能需求工程研究，使课程材料、系统实现、知识库、数据集、基准、实验结果和论文资产能够共同演化。
