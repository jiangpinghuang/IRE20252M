# benchmark/tasks/datasets

本目录保留通用数据集建设材料，服务于基准任务定义和可复现实验。

## 内容范围

- `raw/`：原始数据或原始样例。
- `processed/`：清洗、规范化或可直接评测的数据。
- `schemas/`：数据格式、字段说明和校验规则。
- `annotation-guidelines/`：标注规范、标签定义和质量控制规则。
- `examples/`：小规模示例数据和格式样例。

## 维护规则

- 数据必须记录来源、授权、处理方式和适用任务。
- 小组专属数据放在对应 `groups/G*/datasets/`，通用数据再归档到此处。
- 不提交敏感数据、个人信息或授权不清的材料。
