# benchmark

本目录存放智能需求工程研究的基准任务、评价指标、基线方法和实验结果。

## 内容范围

- `tasks/`：任务定义、输入输出格式、数据集说明和实验配置。
- `metrics/`：自动评价指标、人工评价量表和统计口径。
- `baselines/`：基线方法、脚本和可复现实验入口。
- `results/`：实验结果、分析记录、图表和论文表格。

## 维护规则

- 每个基准任务应明确输入、输出、数据来源、标注方式、指标和基线。
- 结果必须记录数据版本、代码版本、模型或系统配置和运行条件。
- 不提交无法解释来源的结果文件。
- 与某个小组强相关的数据或实验，可同时在 `groups/G*/` 中保留工作副本。
