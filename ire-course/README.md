# IRE Course

This directory organizes graduate course materials for Intelligent Requirements Engineering.

## Course Goals

- Teach research-oriented requirements engineering.
- Connect students to long-term ChatREQ and IRE infrastructure.
- Produce reusable group artifacts for papers, datasets, benchmarks, and system modules.
- Train students to write ICSE/ASE/FSE-style research papers.

## Directory Overview

| Path | Purpose |
| --- | --- |
| `syllabus/` | Course schedule, reading plan, policies, and learning outcomes. |
| `slides/` | Lecture slides and discussion materials. |
| `group-tasks/` | Eight research group task definitions. |
| `weekly-reports/` | Weekly progress reports from student groups. |
| `paper-templates/` | Course-facing paper templates and checklists. |
| `codex-prompts/` | Prompt patterns for research and engineering assistance. |
| `project-guidelines/` | Project rules, deliverables, and grading guidance. |
| `final-reports/` | Final student reports and presentation materials. |
