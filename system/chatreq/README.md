# ChatREQ

ChatREQ 是本仓库的核心系统资产，定位为知识驱动的多智能体需求工程系统。

## 内容范围

- `agents/`：智能体角色、协作逻辑和角色策略。
- `knowledge/`：系统本地知识、检索适配和任务上下文。
- `artifacts/`：生成或整理的需求工程产物。
- `workflows/`：端到端流程和模块级工作流。
- `evaluation/`：系统级评测入口、输出和分析材料。
- `datasets/`：系统样例数据或数据适配器。
- `docs/`：系统设计、接口和使用说明。
- `examples/`：示例项目、输入和输出。
- `prompts/`：智能体提示词和任务指令。
- `configs/`：运行配置和实验配置。
- `tests/`：单元测试、集成测试和回归测试。

## 维护规则

- 新模块应说明输入、输出、依赖知识、失败模式和评价方式。
- 与论文或基准相关的实现应能追溯到对应任务和实验。
- 系统级提示词留在本目录，通用 AI 使用材料放入 `ai-resources/`。
