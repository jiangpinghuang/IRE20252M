# ai-resources

本目录存放 AI 辅助研究、写作和工程协作的通用资源。

## 内容范围

- `prompts/`：通用提示词和任务指令。
- `workflows/`：AI 辅助研究或工程流程。
- `guidelines/`：使用规范、质量要求和检查清单。
- `tutorials/`：教学或上手材料。
- `references/`：工具、模型或方法参考资料。

## 维护规则

- 通用资源放在本目录，ChatREQ 系统内部提示词放在 `system/chatreq/prompts/`。
- 提示词应说明目标任务、输入、输出和适用限制。
- 不提交包含敏感信息的对话或密钥。
