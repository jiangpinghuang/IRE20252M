# IRE-Evaluation

IRE-Evaluation stores shared evaluation assets for Intelligent Requirements Engineering and ChatREQ.

## Scope

This directory supports:

- Metric definitions.
- Evaluation scripts.
- Experiment configurations.
- Result analysis.
- Plots.
- LaTeX-ready tables.

## Evaluation Principles

Evaluation should connect claims to evidence. Each experiment should define the research question, dataset version, baseline systems, metrics, configuration, and result interpretation.

## Core Metrics

- Completeness.
- Consistency.
- Conflict detection precision and recall.
- Traceability quality.
- Feasibility judgment accuracy.
- Human effort reduction.
- Downstream utility.
- Reproducibility.

## Reporting

Results should be exported in formats that support both analysis and paper writing, including machine-readable result files, plots, and LaTeX table fragments.
