# groups

本目录存放八个研究小组的长期工作区。

## 目录规则

每个小组目录包含：

- `paper/`：论文草稿和投稿材料。
- `reports/`：周报、阶段报告和总结报告。
- `artifacts/`：需求产物、标注样例、流程图等研究资产。
- `experiments/`：实验脚本、配置和运行记录。
- `datasets/`：小组数据集和标注材料。
- `figures/`：论文或报告图表。
- `src/`：小组实现代码。
- `notes/`：研究笔记、任务说明和讨论记录。

## 维护规则

- 小组工作优先在本目录闭环，再向 `papers/`、`benchmark/`、`knowledge/` 或 `system/` 汇总。
- 不记录虚假进展或未运行的实验结果。
- 重要结论应能追溯到数据、代码、实验或人工评审记录。
