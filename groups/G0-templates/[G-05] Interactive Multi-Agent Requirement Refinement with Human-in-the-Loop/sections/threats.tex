\section{Threats to Validity}
\label{threats}

\subsection{Internal Validity}
\label{subsec:threats-internal}
Internal validity threats may arise from differences in participant experience, task familiarity, feedback quality, and model response variability. The study should mitigate these threats through consistent task instructions, controlled interaction budgets, and repeated evaluation where feasible.

\subsection{External Validity}
\label{subsec:threats-external}
External validity threats concern whether the findings generalize beyond the selected requirement scenarios, domains, participants, and model settings. The evaluation should therefore include requirements with different ambiguity types, constraint types, and stakeholder goals.

\subsection{Construct Validity}
\label{subsec:threats-construct}
Construct validity threats arise if metrics such as controllability, satisfaction, or refinement effectiveness do not accurately capture the intended concepts. The study should combine expert assessment, user questionnaires, revision trace analysis, and clearly defined scoring rubrics.
