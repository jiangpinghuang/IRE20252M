\section{Threats to Validity}
\label{threats}

\subsection{Internal Validity}
\label{sec:threats-internal}

Internal threats may arise from prompt sensitivity, agent interaction order, evaluator subjectivity, and the quality of human-reviewed reference mappings.
The study should mitigate these threats through fixed protocols, independent review, and documented adjudication.

\subsection{External Validity}
\label{sec:threats-external}

External threats concern whether results generalize across domains, requirement styles, system sizes, and architecture paradigms.
The evaluation should report dataset diversity and avoid claims beyond the studied project scope.

\subsection{Construct Validity}
\label{sec:threats-construct}

Construct threats concern whether the selected metrics faithfully measure requirement-to-architecture alignment.
The study should justify each metric and combine automatic assessment with expert review where necessary.
