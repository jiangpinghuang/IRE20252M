\section{Threats to Validity}
\label{threats}

\subsection{Internal Validity}
Internal validity threats include prompt sensitivity, inconsistent human annotations,
and possible bias introduced by the selected requirement examples. The study should
mitigate these threats through fixed evaluation prompts, clear annotation guidelines,
and independent review of conflict labels.

\subsection{External Validity}
External validity threats arise because requirement conflicts vary across domains,
stakeholder groups, and specification styles. The evaluation should therefore report
the domains and requirement types covered by the dataset and avoid overgeneralizing
from a narrow sample.

\subsection{Construct Validity}
Construct validity threats concern whether the selected metrics capture the intended
notions of conflict detection and consistency verification. Accuracy and false
positive rate should be complemented by explanation quality and human verification
effort to better reflect practical usefulness.
