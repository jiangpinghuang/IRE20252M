\section{Threats to Validity}
\label{threats}

\subsection{Internal Validity}
Internal validity threats may arise from differences in stakeholder responses, analyst judgment, or clarification ordering across sessions.
The study should control initial requirements and use consistent evaluation procedures.

\subsection{External Validity}
External validity threats arise from limited domains, scenario complexity, and participant diversity.
The results should be interpreted as evidence for requirement elicitation scenarios similar to those evaluated.

\subsection{Construct Validity}
Construct validity threats concern whether the selected metrics adequately capture requirement completeness, missing-information coverage, clarification effectiveness, and human effort.
The study should define each metric before annotation and use clear coding guidelines.
