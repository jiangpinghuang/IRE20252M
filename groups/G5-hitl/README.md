# G5 HITL

## 研究主题

面向需求工程的人机协同与自适应人工介入控制。

## 对应Agent

AdaptiveHITL-REQ：负责不确定性估计、风险评分、人工介入选择和反馈记录。

## 当前问题

所有步骤都人工审查成本过高，完全自动化又可能漏掉高风险问题。

## 方法方向

设计自适应介入策略，比较固定阈值、随机抽样、始终人工和从不人工等策略。
