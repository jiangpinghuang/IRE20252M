# G1 Elicitation

## 研究主题

面向缺失信息的动态多轮需求获取。

## 对应Agent

ChatClarify：负责缺失信息识别、问题生成、答案整合和澄清停止判断。

## 当前问题

早期需求常缺少利益相关者、目标、约束、假设、验收条件和环境条件。

## 方法方向

构建缺失信息分类体系，生成多轮澄清问题，并评估每轮信息增益和需求完整性提升。
